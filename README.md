# DeePonyOS ✨

Your magical My Little Pony collection tracker — soft, sparkly, and child-friendly.

**Live:** https://shamikhahmed.github.io/DeePonyOS/

## Features

- 🏠 **Stable** — greeting, pony counter, generation rainbow
- 🦄 **Collection** — photos, filters, search, 5 generations
- 💫 **Wishlist** — dream ponies with priority
- 🗂️ **Shelves** — organise by shelf location
- 🌈 **Stats** — bubbles, achievements, collection health

## Install on iPhone

1. Open in **Safari**
2. Tap **Share** → **Add to Home Screen**
3. Launch from home screen for full-screen native feel

## Develop locally

```bash
python3 -m http.server 8765
# open http://localhost:8765
```

## Deploy (GitHub Pages)

Push to `main` — Pages serves from repo root with `.nojekyll`.
