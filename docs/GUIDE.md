# DeePonyOS — User Guide

DeePonyOS is a **magical My Little Pony collection tracker** — soft, sparkly, and child-friendly. Track ponies, wishlists, shelves, and collection stats across five generations.

**Live app:** https://shamikhahmed.github.io/DeePonyOS/

---

## Getting started

1. Open DeePonyOS in Safari on iPhone or iPad.
2. Watch the splash animation — onboarding appears on first visit.
3. Set your collector name and favourite generation.
4. Land on **Stable** — your home screen with pony count and rainbow.
5. Install: **Share → Add to Home Screen** for full-screen PWA.

---

## Navigation

| Tab | Purpose |
|-----|---------|
| **Stable** | Greeting, pony counter, generation rainbow, anniversary banner |
| **Collection** | All ponies — photos, filters, search, 5 generations |
| **Wishlist** | Dream ponies with priority (Must / Want / Someday) |
| **Shelves** | Physical shelf locations for organisation |
| **Stats** | Bubbles, achievements, generation checklist, collection health |

---

## Adding a pony

1. Tap the **+** floating button (or Add from Collection).
2. Fill in name, generation, type, and condition.
3. Add a photo from camera or gallery.
4. Optional: shelf location, acquisition date, notes.
5. Save — pony appears in Collection and updates Stable counter.

---

## Generations

DeePonyOS supports five MLP generations with colour-coded pills:

- G1 · G2 · G3 · G4 · G5

Filter Collection by generation using the chip row. Stats shows a **generation checklist** — unique names owned vs the pony name database per gen.

---

## Wishlist

Track ponies you don't own yet:

- **Must** — top priority (red border)
- **Want** — nice to have (yellow)
- **Someday** — future dreams (mint)

Move items to Collection when acquired.

---

## Shelves

Organise by physical location:

- Create shelf names (e.g. "Pink Shelf", "Box 3")
- Assign ponies to shelves from edit screen
- Browse Shelves tab to see what's where

---

## Stats & achievements

- **Bubble chart** — visual breakdown by generation
- **Achievements** — milestones for collection size and variety
- **Collection health** — completeness score per generation
- **Anniversary banner** — ponies acquired on this day in prior years appear on Stable

---

## Modes

| Mode | How to enable |
|------|---------------|
| **Collector mode** | Settings — deeper purple theme for serious collectors |
| **Dark mode** | Settings — night-friendly dark cards |
| **Child-friendly** | Default — soft pink, rounded, large tap targets |

---

## Data & privacy

- All data stored in `localStorage` on your device
- No accounts, no cloud sync
- Photos stored as base64 in local storage — export device backups via iCloud if needed
- Works offline after first load

---

## Tips

- Log acquisition dates — unlocks anniversary banners on Stable.
- Use generation checklist in Stats to find gaps in your collection.
- Wishlist photos help remember which variant you want.
- Collector mode is easier on the eyes for long cataloguing sessions.

---

## Troubleshooting

| Issue | Fix |
|-------|-----|
| Photos won't save | Check storage quota; reduce photo size before upload |
| Count wrong | Refresh by switching tabs; check filters aren't active |
| Onboarding loops | Clear site data and revisit — or complete all onboarding steps |
